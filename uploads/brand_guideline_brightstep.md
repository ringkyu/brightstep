## 1. 브랜드 철학

### 브랜드 에센스 (Brand Essence)

```
  ╔══════════════════════════════════════════════╗
  ║                                              ║
  ║   "다음 걸음을 밝히는 실전의 파트너"           ║
  ║   The Practical Partner for the Next Step    ║
  ║                                              ║
  ╚══════════════════════════════════════════════╝
```

브라이트스텝 컨설팅의 브랜드는 세 가지 감각 위에 세워집니다.

**현장의 온기 (Warmth of the Field)**
우리의 고객은 화려한 오피스빌딩이 아니라 골목 카페, 공단 공장, 지역 학원에서 하루를 보냅니다. 브랜드는 그 현장의 온기를 담아야 합니다. 딱딱한 컨설팅 회사가 아니라, 함께 걷는 동반자처럼 느껴져야 합니다.

**신뢰의 단단함 (Solidity of Trust)**
소상공인에게 컨설팅 비용은 결코 작은 결정이 아닙니다. 브랜드 전반에서 전문성과 안정감이 묻어나야 합니다. 가볍거나 트렌디해 보이는 것보다, 믿을 수 있다는 인상이 먼저입니다.

**행동의 명확함 (Clarity of Action)**
'분석'보다 '실행', '보고서'보다 '결과'. 브랜드 커뮤니케이션은 항상 명확한 행동과 변화를 중심에 둡니다. 모호한 표현, 추상적인 슬로건은 사용하지 않습니다.

---

### 브랜드 퍼스낼리티 (Brand Personality)

| 속성 | 설명 | 반대 극단 (지양) |
|------|------|-----------------|
| **신뢰할 수 있는** | 약속을 지키고, 정보를 투명하게 공유한다 | 과장되거나 모호한 |
| **실용적인** | 이론보다 현장, 말보다 실행을 앞세운다 | 화려하거나 추상적인 |
| **따뜻한** | 고객의 상황을 공감하고 곁에 선다 | 냉정하거나 거리를 두는 |
| **명확한** | 복잡한 것을 단순하게 전달한다 | 전문용어 남발, 복잡한 |
| **성장하는** | 고객과 함께 발전한다는 에너지를 품는다 | 정체되거나 보수적인 |

## 2. 브랜드 포지셔닝

### 포지셔닝 스테이트먼트

> **브라이트스텝 컨설팅은**
> 서울·인천·강원·대전의 소상공인 및 중소기업을 위한
> **현장 밀착형 경영 컨설팅 파트너**로서,
> 이론이 아닌 실행 중심의 솔루션과
> 지속적인 성과 모니터링을 통해
> 고객이 스스로 성장하는 역량을 키울 수 있도록 돕습니다.

### 경쟁 포지셔닝 맵

```
                     ↑ 고관여 / 전략적
                     │
     대형 컨설팅 ─────┤───── (프리미엄, 대기업 중심)
     (McKinsey 류)    │
                     │
  ◀──── 거리감 ───────┼──────── 밀착감 ────▶
                     │
                     │       ★ BrightStep
                     │         (현장 밀착, 중소·소상공인 전문)
                     │
     세무·법무사 ──────┤───── (단발성, 규정 준수 중심)
                     │
                     ↓ 저관여 / 단발성
```

## 3. 로고

### 3-1. 로고 컨셉

로고는 두 요소로 구성됩니다.

- **심볼 마크**: 위를 향해 오르는 계단 형태의 'B' 이니셜 — 성장과 다음 걸음을 상징
- **워드마크**: "BrightStep" + "Consulting" 한/영 병기

```
  ┌─────────────────────────────────────────────┐
  │                                             │
  │   [B↑]  BrightStep                         │
  │         브라이트스텝 컨설팅                   │
  │                                             │
  └─────────────────────────────────────────────┘

  심볼:  계단을 오르는 화살표가 내장된 'B' 블록형 심볼
  서체:  워드마크 전용 커스텀 세미볼드
  비율:  심볼 : 워드마크 = 1 : 3.5
```

### 3-2. 로고 버전 체계

| 버전 | 용도 | 배경 |
|------|------|------|
| **Full Logo (가로형)** | 공식 문서, 웹 헤더, 명함 | 흰색 / 밝은 배경 |
| **Full Logo (세로형)** | SNS 프로필, 정사각 영역 | 흰색 / 밝은 배경 |
| **심볼 단독** | 앱 아이콘, 파비콘, 소형 각인 | 모든 배경 |
| **역전 버전 (White)** | 다크 배경 / 컬러 배경 위 사용 | 브랜드 네이비 / 사진 위 |
| **모노크롬** | 흑백 인쇄, 도장, 서류 인장 | 흰색 / 회색 배경 |

### 3-3. 로고 보호 영역 (Clear Space)

```
  ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐
                X
  │   ┌───────────────────┐   │
    X │  [B↑] BrightStep  │ X
  │   └───────────────────┘   │
                X
  └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘

  X = 심볼 마크 높이의 1/2
  보호 영역 안에 다른 텍스트, 이미지, 선이 침범해서는 안 됩니다.
```

### 3-4. 최소 사용 크기

| 매체 | 최소 너비 |
|------|-----------|
| 인쇄물 (전체 로고) | 30mm |
| 디지털 (전체 로고) | 120px |
| 심볼 단독 (인쇄) | 8mm |
| 심볼 단독 (디지털) | 32px |

## 5. 타이포그래피

### 5-1. 서체 체계

브라이트스텝은 **한국어와 영어 환경 모두를 고려한** 이중 서체 체계를 사용합니다.

#### 한국어 서체

| 역할 | 서체명 | 굵기 | 용도 |
|------|--------|------|------|
| **Display (제목)** | Noto Serif KR | Bold (700) | 슬로건, 대형 헤드라인, 인용구 |
| **Heading (소제목)** | Pretendard | SemiBold (600) | 섹션 제목, 카드 제목 |
| **Body (본문)** | Pretendard | Regular (400) | 본문 텍스트, 설명 |
| **Caption (캡션)** | Pretendard | Light (300) | 주석, 출처, 소형 레이블 |

#### 영문 서체

| 역할 | 서체명 | 굵기 | 용도 |
|------|--------|------|------|
| **Display** | Playfair Display | Bold | 영문 헤드라인, 로고 보조 |
| **Heading / Body** | Inter | SemiBold / Regular | 영문 UI 텍스트, 데이터 |
| **Data / Code** | JetBrains Mono | Regular | 수치, 통계, 코드 |

> Noto Serif KR은 신뢰와 격식을 부여하며, Pretendard는 현대적 가독성을 담당합니다. 두 서체의 조합은 '전통적 신뢰 + 현장 친화적 명확함'이라는 브랜드 퍼스낼리티를 타이포그래피로 구현합니다.

## 6. 이미지 & 사진 스타일

### 6-1. 사진 방향성

브라이트스텝의 사진은 **'현장에 있는 진짜 사람'**을 담아야 합니다.

**선호하는 이미지**
- 실제 매장·사무실·공장 현장에서 촬영된 사진
- 컨설턴트와 사업주가 함께 자료를 검토하는 장면
- 지역 상권의 생동감 있는 모습 (골목, 시장, 공단)
- 사업주의 표정에 자신감과 안도감이 담긴 장면
- 자연광 또는 따뜻한 실내광 활용

**지양하는 이미지**
- 과도한 리터칭 / 스튜디오 합성 느낌
- 외국인 모델 또는 글로벌 대도시 배경
- 어둡거나 불안감을 주는 색조
- 클립아트, 셔터스톡 티 나는 이미지
- 손 모아 악수, 인포그래픽 구·화살표 과다 사용

### 6-2. 사진 후보정 가이드

| 항목 | 방향 |
|------|------|
| **색온도** | 5,000–5,500K (자연광 계열, 따뜻하되 과도하지 않게) |
| **채도** | 원본 대비 +5~10% (과포화 금지) |
| **밝기** | 중간~밝음 (어두운 이미지 지양) |
| **비네팅** | 사용 금지 |
| **필터** | 브랜드 LUT 적용 (네이비-골드 색조 보정) |

### 6-3. 이미지 오버레이

텍스트가 사진 위에 올라갈 경우:

```
  옵션 A: 스텝 네이비 오버레이 (불투명도 55–70%)
  옵션 B: 하단→상단 그라디언트 (네이비 80% → 투명 0%)
  옵션 C: 골드 좌측 세로 바(bar) + 네이비 오버레이 조합
```

흰색 텍스트를 사진 위에 그대로 올리는 방식은 금지합니다.

## 9. 브랜드 보이스 & 톤

### 9-1. 브랜드 보이스 (변하지 않는 것)

브라이트스텝의 커뮤니케이션은 항상 이 세 가지 목소리를 가집니다.

**현장을 아는 전문가**
경험에서 나온 언어를 씁니다. 교과서 표현보다 사업주가 일상에서 쓰는 말을 선택합니다.

**솔직한 동반자**
좋은 소식만 전하지 않습니다. 문제를 직접 짚고, 해결책을 분명히 말합니다.

**행동을 이끄는 안내자**
모든 커뮤니케이션은 다음 행동으로 이어집니다. "생각해보세요"가 아닌 "지금 연락하세요"입니다.
